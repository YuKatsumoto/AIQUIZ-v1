# Phase 1 超詳細手順書

**前提**: Unreal Editor の言語は**日本語**。毎回「どこをクリックするか」「何を入力するか」を 1 ステップずつ書いています。

---

## 手順 1: プロジェクトを開く

| ステップ | 操作 | 補足 |
|---------|------|------|
| 1-1 | エクスプローラーでフォルダを開く | 例: `C:\Users\kykat\Desktop\AIQUIZ\AIQuizUE\AIQUIZUE` |
| 1-2 | **AIQUIZUE.uproject** をダブルクリック | 左クリックで 2 回素早く押す |
| 1-3 | エディタが起動するまで待つ | 初回は 1〜3 分かかることがある |

---

## 手順 2: C++ をビルドする

### 方法 A: エディタから

| ステップ | 操作 | 補足 |
|---------|------|------|
| 2A-1 | 画面一番上のメニューバーを確認 | 「ファイル」「編集」「表示」などが並んでいる |
| 2A-2 | **ツール** をクリック | メニューバー内の「ツール」を左クリック |
| 2A-3 | **コンパイル** をクリック | ドロップダウン内の「コンパイル」を左クリック |
| 2A-4 | ビルドが終わるまで待つ | 右下の進捗表示を確認。数十秒〜数分 |
| 2A-5 | 「コンパイルが完了しました」などと出れば成功 | エラーが出た場合は後述のトラブルシューティングを参照 |

**ショートカット**: **Ctrl + Alt + F11** でもコンパイル可能。

### 方法 B: Visual Studio から（エディタを閉じてから）

| ステップ | 操作 | 補足 |
|---------|------|------|
| 2B-1 | Unreal Editor を終了 | ファイル → 終了、またはウィンドウの ✕ |
| 2B-2 | エクスプローラーで `.uproject` があるフォルダを開く | AIQUIZUE.uproject がある場所 |
| 2B-3 | **AIQUIZUE.uproject** を右クリック | 右クリックメニューを開く |
| 2B-4 | **Visual Studio 用プロジェクトファイルを生成** をクリック | メニュー内のその項目を左クリック |
| 2B-5 | しばらく待つ | 「生成しました」などのメッセージが出る |
| 2B-6 | 同じフォルダ内の **AIQUIZUE.sln** をダブルクリック | Visual Studio が起動する |
| 2B-7 | 上部の構成で **Development Editor** を選択 | プルダウンで選ぶ |
| 2B-8 | 右隣のプラットフォームで **Win64** を選択 | プルダウンで選ぶ |
| 2B-9 | メニュー **ビルド** → **ソリューションのビルド** をクリック | またはキーボード **F7** |
| 2B-10 | 下部「出力」に「ビルド succeeded」と出れば成功 | エラーが出た場合はメッセージを確認 |
| 2B-11 | Visual Studio を閉じてよい | その後 AIQUIZUE.uproject をダブルクリックしてエディタを開く |

## 手順 3: ウィジェット ブループリント（WBP_Quiz）を作成

| ステップ | 操作 | 補足 |
|---------|------|------|
| 3-1 | 画面下部の **コンテンツ ブラウザ** を確認 | フォルダツリーとアセット一覧があるパネル |
| 3-2 | コンテンツ ブラウザ内の**何もない灰色のところ**で右クリック | フォルダ内の空きスペースで右クリック |
| 3-3 | メニューから **ユーザーインターフェース** にマウスを乗せる | サブメニューが開く |
| 3-4 | **ウィジェット ブループリント** を左クリック | 新規ウィジェット作成ダイアログが開く |
| 3-5 | 名前を `WBP_Quiz` と入力 | 既にフォルダが選択されていればそのままで可 |
| 3-6 | **エンター** を押すか、**保存** などをクリック | アセットが作成され、ウィジェット エディタが開く |
| 3-7 | ウィジェット エディタが開いたら、右上あたりの **クラスの設定** を探す | 歯車アイコン、または「クラスの設定」という文字 |
| 3-8 | **クラスの設定** をクリック | 詳細パネルにクラス設定が表示される |
| 3-9 | **親クラス** の右側（現在「UserWidget」などと書いてあるところ）をクリック | プルダウンかクラス選択ボタン |
| 3-10 | **すべてのクラスを表示** にチェックを入れる | 検索しやすくなる |
| 3-11 | 検索欄に `QuizWidget` と入力 | 候補が絞り込まれる |
| 3-12 | 一覧から **Quiz Widget** をクリックして選択 | C++ の QuizWidget クラス |
| 3-13 | **Ctrl + S** を押して保存 | または ファイル → 保存 |
| 3-14 | タブを閉じるか、コンテンツ ブラウザに戻る | WBP_Quiz の作成は完了 |

**補足**: テキストを手動で追加する必要はありません。C++ が自動作成します。

## 手順 4: 壁ブループリント（BP_QuizWall）を作成

| ステップ | 操作 | 補足 |
|---------|------|------|
| 4-1 | コンテンツ ブラウザで右クリック | フォルダ内の空きスペース |
| 4-2 | **ブループリント クラス** を左クリック | 親クラス選択ダイアログが開く |
| 4-3 | 左下の **すべてのクラスを表示** にチェック | チェックボックスをクリック |
| 4-4 | 上部の検索欄に `QuizWall` と入力 | 候補が絞り込まれる |
| 4-5 | 一覧から **QuizWall** をクリック | C++ の QuizWall クラス |
| 4-6 | **選択** ボタンをクリック | ダイアログが閉じ、アセット作成画面になる |
| 4-7 | 名前を `BP_QuizWall` と入力 | 必要ならフォルダも選択 |
| 4-8 | エンターを押すか保存 | BP_QuizWall が作成され、ブループリント エディタが開く |
| 4-9 | 左側の **コンポーネント** パネルで **WallMesh** をクリック | スタティックメッシュ コンポーネント |
| 4-10 | 右側の **詳細** パネルで **スタティックメッシュ** の右側（「なし」など）をクリック | メッシュ選択ウィンドウが開く |
| 4-11 | 検索欄に `Cube` や `立方体` と入力 | 基本形状を探す |
| 4-12 | **Cube**（立方体）を選択してクリック | 壁に立方体が設定される |
| 4-13 | 詳細パネルの **トランスフォーム** または **スケール** を展開 | あれば |
| 4-14 | **スケール** を X=0.05, Y=8, Z=4 程度に設定 | 薄い横長の壁になる |
| 4-15 | **Wall Speed** を探し、200 など好みの値に | 壁の移動速度 |
| 4-16 | **WallBodyCollision** を選択し、詳細で **コリジョン プリセット** を **OverlapAllDynamic** に | プレイヤーと反応するように |
| 4-17 | **DoorLeftCollision** を選択し、同様に **OverlapAllDynamic** に | 同上 |
| 4-18 | **DoorRightCollision** を選択し、同様に **OverlapAllDynamic** に | 同上 |
| 4-19 | **Ctrl + S** で保存 | 壁ブループリント完了 |

## 手順 7: 廊下レベルを作成

| ステップ | 操作 | 補足 |
|---------|------|------|
| 7-1 | メニュー **ファイル** → **新規レベル** をクリック | または **基本** を選ぶ |
| 7-2 | 「空のレベル」または「基本」を選択して開く | 何もないレベルが開く |
| 7-3 | メニュー **ファイル** → **現在のレベルを保存**（Ctrl+S） | 名前を `Lvl_Quiz` などにする |
| 7-4 | 左側の **パレット**（プレイス モード）で **基本** を展開 | 床や壁を置くためのプリミティブ |
| 7-5 | **立方体**（Cube）をレベルにドラッグ | 床用。複数置いて廊下の形に |
| 7-6 | 立方体の **スケール** を X=20, Y=4, Z=0.2 などに | 薄く横長の床 |
| 7-7 | 左右に壁用の立方体を並べる | 廊下の形になるように |
| 7-8 | **プレイヤー スタート** をパレットからドラッグして配置 | 「検索」で「プレイヤー」と入力すると出る |
| 7-9 | プレイヤー スタートを廊下の**手前**（原点付近）に置く | 奥（+X）を向くように回転 |
| 7-10 | コンテンツ ブラウザから **BP_QuizWall** をレベルにドラッグ | 廊下の**奥**（+X 方向）に置く |
| 7-11 | BP_QuizWall の位置を調整 | 例: X=2000, Y=0, Z=0 付近（廊下の奥） |
| 7-12 | BP_QuizWall の回転を調整 | 壁の前面がプレイヤー（-X）を向くように |
| 7-13 | **Ctrl + S** でレベルを保存 | レベル作成完了 |

## トラブルシューティング（詳細）

| 症状 | 確認すること |
|------|-------------|
| ビルドエラー「HTTP」「Json」 | AIQUIZUE.Build.cs に "HTTP", "Json", "JsonUtilities" があるか確認 |
| クラスがコンテンツ ブラウザに出ない | エディタを閉じて VS でフルビルド → エディタを開き直す |
| 問題が表示されない | WBP_Quiz の親クラスが QuizWidget か確認 |
| 壁が動かない | レベルに BP_QuizWall が 1 つあるか、ワールド設定のゲームモードが BP_QuizGameMode か確認 |
| 当たり判定が反応しない | 壁の 3 つのボックス コリジョンのプリセットを OverlapAllDynamic に |
| API に接続できない | サーバー起動、URL が http://127.0.0.1:8000 か（失敗時はフォールバック問題でプレイ可能） |
